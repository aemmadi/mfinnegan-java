Data Types

To make variables in Java, they need a data type. A data type tells the Java compiler what kind of data a variable is.

Integers

To declare integer variables: 
int a = 0;
If you want to change the value of a variable: 
a = 20;
The "int" primitive type can only store integers. The following would not work: 
a = 5.6;

Decimals

To declare variables that store decimals : 
double d = 4.5;
The following code will output 4 : 
System.out.println(4.0);
Another way to declare decimals is to use "float" : 
float f = 4.5f;
Floats must have an "f" letter at the end of the decimal.

Strings

To store a series of characters in a variable, use "String" data type : 
String sentence = "hello world";
Strings are made up of characters, so you can access each of their characters through their index. Indexes always start at 0. The following code will print "h": 
System.out.println(sentence.charAt(0));
When 2 or more strings are added together, that is called "Concatenation" : 
String word1 = "hello";
String word2 = "world";
String sentence = word1 + word2;
