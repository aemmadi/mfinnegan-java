Operators

To use Math.functions in java, you need to be familiar with the math operators present

Arithmetic Operators

Addition :
To add ints or doubles use the operator "+" to indicate what variables need to be added. 
int numOne = 20;
int numTwo = 10; 
System.out.print(numOne + numTwo);
Output : 30
Subtraction :
To subtract ints or doubles use the operator "-" to indicate what variables need to be subtracted. 
int numOne = 20;
int numTwo = 10;
System.out.print(numOne - numTwo);
Output : 10
Multiplication :
To multiply ints or doubles use the operator "*" to indicate what variables need to be multiplied. 
int numOne = 20;
int numTwo = 10;
System.out.print(numOne * numTwo);
Output : 200
Division :
To divide ints or doubles use the operator "/" to indicate what variables need to be divided. 
int numOne = 20;
int numTwo = 10;
System.out.print(numOne / numTwo);
Output : 2
